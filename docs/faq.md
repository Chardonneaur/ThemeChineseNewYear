## FAQ

### How do I activate the theme?

Go to **Administration > Platform > Themes** and click **Activate** next to "ThemeChineseNewYear".

### Does this theme modify any data or tracking?

No. This is a purely cosmetic theme. It only changes colors and adds a background pattern. It does not affect tracking, data collection, or any Matomo functionality.

### Which Matomo versions are supported?

Matomo 5.x is supported.

### Can I use this alongside other plugins?

Yes. Since this is a theme, it only affects the visual appearance. All plugins remain fully functional.
