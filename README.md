# Theme Chinese New Year

A festive Matomo theme inspired by Chinese New Year celebrations. Features a red and gold color palette with a subtle repeating background pattern of stylized lanterns and auspicious clouds.

## Features

- Red and gold color scheme throughout the interface
- Deep red header with gold text
- Subtle SVG background pattern with lanterns and clouds
- Graph colors in Chinese red, imperial gold, jade green, and lantern orange
- Clean white cards over the patterned background

## Installation

Place in your Matomo `plugins/` directory and activate via Administration > Platform > Themes.
